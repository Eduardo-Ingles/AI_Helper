1. Input:  
   - Fornisci un file sorgente con le colonne denominate `newXXX`.  
 
2. Colonne Utilizzate:  
   - `rawRegister`: compilare solo se presente INDIRIZZO (NON SPLITTARE INDIRIZZO . BITINDEX)
   - `rawBitIndex`: compilare solo se presente BITINDEX  (NON SPLITTARE INDIRIZZO . BITINDEX)  
   - `rawAddress`: compilare solo se presente INDIRIZZO . BITINDEX già nella colonna  
   - `rawRegisterType`  
   - `rawDataType`  
   - `rawDispositivo`  
   - `rawSegnale`  
   - `rawDescrizione`  
   - `rawDescr&Signal`  
   - `rawLogica`  
   - `rawTags`  
   - `rawQuadro`  
   - `rawCabina`  
   - `rawDirezione`  
   - `rawReference`  
   - `rawIpPlc`  
   
   Colonne pronte per copia > incolla > testo in colonna:
      rawRegister,rawBitIndex,rawAddress,rawRegisterType,rawDataType,rawDispositivo,rawSegnale,rawDescrizione,rawDescr&Signal,rawLogica,rawTags,rawQuadro,rawCabina,rawDirezione,rawReference,rawIpPlc

3. Creazione della Pseudo Mappa:  
   - Per ogni colonna `raw`, utilizza una formula semplice:  
     ```excel
     =SE(CELLARIF != ""; CELLARIF; "") ===> se la cella selezionata non è vuota, allora riferimento al valore in quella cella, alrimenti lascia stringa vuota
     ```  
     oppure copia direttamente il valore.
 
4. Trattamento dei Dati:  
   - Il codice elabora i dati in background, escludendo quelli non essenziali.  
   - Non sono accettati valori contrastanti come:  
     ```plaintext
     "Ventilatore sud canna nord direzione sud"
     ```  
     ⚠️
     In tal caso, risolvi il problema con 𓀡:  
     - CTRL + A → Seleziona tutto  
     - CTRL + DEL → Cancella  
     - ALT + F4 → Chiudi
     ⚠️
